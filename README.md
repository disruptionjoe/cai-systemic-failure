# CAI Systemic Failure

This repository makes systemic failure legible without manufacturing certainty.
It maintains the Failure Topology, compares prior art, preserves contradictory
evidence, forms answerable inquiries, and routes work to the right sovereign owner.

Current posture: public research source; founder-led; CAI-directed automation
owner-permitted only under the explicit live CapacityOS System grant; no human
or field research. Hypotheses are hypotheses, negative evidence remains visible,
and a map is not an intervention.

Start with `STATUS.md`, `GOVERNANCE.md`, `LANES.yaml`, `topology/README.md`,
`prior-art/README.md`, and `inquiries/README.md`.
