# Failure Topology

This is the evidence-bearing relational map of mechanisms, actors, affected
systems, causal links, evidence, counterevidence, interventions, failures, and
open questions. It is not a claim that one universal topology exists.

No schema is created until an actual record is ready to validate it. Every
record must preserve sources, revisions, uncertainty, alternatives, and a
correction route.
